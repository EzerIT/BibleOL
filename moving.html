Warning

For several years, Bible Online Learning has run on a server belonging to the Danish Bible Institute. Further development of Bible OL will be carried out by the NIDA Institute, and this means that the program must be moved to a new server.

The move will probably take a few days. On 8 September 2023 we will make a copy of all data on the current server and move that to the new server. This means that anything that is changed on the old server on or after that date will not be transferred. This includes new user accounts, new classes, new exercises and not least statistics about exercises you make.

When the new server is ready, you will be able to continue work there with the data you had on 8 September.
