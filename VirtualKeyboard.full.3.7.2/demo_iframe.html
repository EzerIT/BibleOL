Installation

  1. Add <script type="text/javascript" src="vk_iframe.js" ></script> to the page's HEAD section
  2. Add container to place keyboard <div id="keyboard"></div>
  3. Add button to open the keyboard <button onclick="IFrameVirtualKeyboard.toggle('default_input_id', 'keyboard');">Keyboard</button>
  4. Add attribute onfocus="IFrameVirtualKeyboard.attachInput(this)" to any fields which should receive keyboard input
  5. Click on the field to open the keyboard
